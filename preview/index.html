Icons

We have a core library of icons from the Symbol Set collection. These icons are technically a live font, allowing designers to manipulate each element the same way any other markup element can be styled with CSS. Furthermore, because the icon font is a vector-based element, the Symbol Set icons are scalable and lightweight, unlike sprite images.

While there are other icon fonts available (Foundation Fonts and Font Awesome), we suggest using Symbol Set for its superior accessibility. Symbols are called with common terms, allowing screen readers and search engines to see words while other users see the symbols.

Reveal makes these very easy to summon and dismiss. The close button is simply an anchor with a unicode character icon and a class of close-reveal-modal. Clicking anywhere outside the modal will also dismiss it.

You can control the width of this modal with classes. Your options are tiny (30%), small (40%), medium (70%), xlarge (95%).

Finally, if your modal summons another Reveal modal, the plugin will handle that for you gracefully.

Tooltips

Tooltips work cross browser and cross platform and are easily added to a page with a simple markup and class structure. You can apply the has-tip class to any element. The tooltips can be positioned on the "tip-bottom", which is the default position, "tip-top" (hehe), "tip-left", or "tip-right" of the target element by adding the appropriate class to them. You can even add your own custom class to style each tip differently. On a small device, the tooltips are full width and bottom aligned.
